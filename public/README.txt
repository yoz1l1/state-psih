HOW TO ADD YOUR PHOTOGRAPHS
============================

1. Put your image files in THIS folder (public/).

2. Name them exactly:
      staff1.jpg
      staff2.jpg
      staff3.jpg
      staff4.jpg
      staff5.jpg
      staff6.jpg
      staff7.jpg
      staff8.jpg
      staff9.jpg
      staff10.jpg

   staff1.jpg  -> first slot  (Richard S. Blackmore, M.D.)
   staff2.jpg  -> second slot (Helena V. Cross, R.N.)
   ...and so on through staff10.jpg.

3. That's it. Refresh the page — the photos appear in the album.
   Any slot without a file shows a "No Photograph on File" plate.

TIPS
----
- Best shape: portrait, roughly 4:5 (e.g. 600 x 750 pixels).
- Formats: .jpg or .webp work best. Keep each file under ~300 KB.
- The vintage sepia/halftone look is applied automatically —
  a plain colour photo will still look period-correct.
- To change the names/captions under each photo, edit src/staff.ts.
